## Description

MatterSnooks is a 2D snooker game built using p5.js for presentation and Matter.js as the physics engine

## Getting Started

### Dependencies

* python

### Executing program
1. Clone the repo
```
git clone https://github.com/NeoXavier/mattersnookz.git
```

2. Run the app
```
python run.py
```
